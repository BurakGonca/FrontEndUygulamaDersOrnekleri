# FrontEnd Uygulama Ders Ornekleri
HTML, CSS, JS ve React çalışma örneklerimin bulunduğu repo.

## Önemli React Projelerim

### <span style="color:red;">JustRecipe</span>

JustRecipe, tariflerinizi yönetebileceğiniz ve paylaşabileceğiniz modern bir web uygulamasıdır. Kullanıcı dostu arayüzü ve zengin özellikleri ile yemek tariflerinizi dijital ortamda organize etmeye elverişlidir. Ziyaretçi olarak tariflere erişebilir, giriş yaparak tarif ekleme, silme ve güncelleme gibi işlemleri gerçekleştirebilirsiniz.

Bu proje, React ve çeşitli yardımcı kütüphaneler kullanılarak geliştirilmiştir.

#### Kullanılan Teknolojiler

- **React**: Kullanıcı arayüzlerini oluşturmak için kullanılan popüler bir JavaScript kütüphanesi.
- **React DOM**: React ile birlikte kullanılan DOM işleme kütüphanesi.
- **React Router**: Tek sayfa uygulamalarında gezinme yönetimi sağlar.
- **Axios**: HTTP istemci kütüphanesi, API isteklerini yönetmek için kullanılır.
- **React Toastify**: Kullanıcıya bildirimler göstermek için kullanılır.
- **React Icons**: Popüler ikon paketlerini projede kullanmak için kullanılır.
- **SweetAlert2**: Güzel, responsive ve özelleştirilebilir alert kutuları oluşturmak için kullanılır.
- **Sass**: Daha verimli ve yapılandırılmış CSS yazmak için kullanılır.
- **Json-Server**: Hem mock API'lar oluşturmak hem de basit ve hızlı RESTful API'lar kurmak için kullanılır.
